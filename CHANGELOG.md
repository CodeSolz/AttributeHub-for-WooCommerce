### Version: 1.0.0 ( June 19, 2026 ) ###
* **New:** Attribute Scanner — detects abbreviated, ugly, and duplicate attribute values across all WooCommerce attributes
* **New:** Master Attribute Directory — create clean master labels and map multiple backend values to a single display label
* **New:** Mapping Editor — drag-and-drop interface for assigning raw attribute values to master groups
* **New:** Clean Frontend Filters — layered navigation shows master labels instead of raw supplier codes
* **New:** Smart Filter Query Expansion — filtering by a master label (e.g., "Black") automatically includes all mapped raw values (BK, BLK, BCK, MULTIBK)
* **New:** Preview Mode — compare before/after filter appearance before going live
* **New:** Product Metabox — shows per-attribute mapping status on the WooCommerce product edit screen
* **New:** Hide Unmapped Values — optionally suppress unmapped or ugly values from frontend filters
* **New:** Duplicate Detection — groups near-identical attribute values for batch mapping
* **New:** CSV Export — export mapping configuration for backup or staging workflows
* **New:** WooCommerce HPOS Compatible — works with High Performance Order Storage
* Initial release
